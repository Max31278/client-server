addVM 	Добавить виртуальную машину
	add name description path
		[name] 	Имя виртульной машины. Задается только с помощью букв и цифр.
		[description]	Описание виртуальной машины
		[path] 	Путь к образу.
		Возможные ошибки:
		1: Имя задано некорректно
		2: Образа по заданному пути не найдено. 
remove	Удаление виртуальной машины	 
	remove name
		[name]	Имя виртуальной машины
		Возможные ошибки:
		1: Имя задано некорректно
list	Вывести список виртульных машин  
	list number
		[number]  Число виртуальных машин, которые нужно вывести. Если параметр не задан,
			  выводит все виртуальные машины. Если число больше, чем количество 
			  виртуальных машин, то выводится все виртуальные машины, что есть.
		Возможные ошибки:
		1: Число введено некоректно
get	Вывести подробную информация о виртуальной машине  
	get name
		[name]	Имя виртуальной машины
		Возможные ошибки:
		1: Имя задано некорректно
		2: Виртуальной машины по заданному имени не существует

exit 	Выход из программы.
